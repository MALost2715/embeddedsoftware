주제: “태양광 기반 자동 급수 시스템 – Solar Water Saver”

1. 작품 개요 (핵심 아이디어)

전기가 부족한 지역에서도 작동 가능한 태양광 전원 기반의 자동 급수 장치를 설계한다.
아두이노와 토양습도 센서, 미니 워터펌프를 활용하여 토양이 건조할 때만 자동으로 물을 공급하는 시스템이다.

2. 주요 특징 및 장점

☀️ 태양광 전원 – 전기 인프라가 부족한 농촌·개도국에서도 사용 가능

💦 자동 급수 제어 – 토양 습도센서가 일정 이하로 떨어질 때만 워터펌프 작동

🔋 저전력 설계 – 배터리 충전식 태양광 모듈로 낮엔 충전, 밤에도 사용 가능

🌍 적정기술 지향 – 값싼 부품으로 누구나 제작 가능

📶 IoT 확장성 – Wi-Fi 모듈 추가 시 클라우드 농업 데이터 관리 가능

3. 대상 사용자

전력 인프라가 부족한 농촌 또는 저개발국가의 농부

물 절약형 자동화 기술이 필요한 농업 현장

학교나 교육용 프로젝트 (STEM, IoT, 환경 교육용)

4. 문제 인식 및 해결 방안
문제점	해결책
전기 없는 지역의 관개 어려움	태양광 패널로 자가 전력 공급
물 낭비 및 농작물 피해	토양 상태 기반 자동 급수
유지보수 어려움	간단한 구조와 저비용 센서 사용
5. 시스템 구성도
   
[ Solar Panel ] ─> [ Battery + Charge Controller ]
                              │
[ Soil Moisture Sensor ] ──> Arduino UNO ──> [ Relay Module ] ──> [ Water Pump ]

6. 사용 부품

Arduino UNO

Soil Moisture Sensor

5V Relay Module

Mini Water Pump

Solar Panel (5V~9V)

Rechargeable Battery Pack

Tubing, Water Tank

7. 주요 작동 원리

센서가 토양 습도를 실시간 측정

일정 기준 이하로 떨어지면 아두이노가 릴레이를 통해 워터펌프 작동

토양이 충분히 젖으면 자동으로 펌프 정지

전력은 태양광 모듈이 낮 동안 충전하여 배터리에 저장

8. 차별화 포인트

전기 없이도 완전 자율 작동

유지비 거의 0원, 설치 간단

환경 친화적 (재생에너지 기반)

지역 맞춤형 확장 가능 (센서 개수, 물탱크 크기 조절 등)

9. 확장 아이디어

블루투스/와이파이 연결 → 스마트폰 앱에서 수분 상태 확인

ESP32 기반 IoT 농장 데이터 수집

물 부족 경고 알림 기능 추가
